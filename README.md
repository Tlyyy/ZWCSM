## 跨设备同步（无服务器）

有两种方式：

### 方式1：无 Token（推荐给你）
1. 在「同步」页点击 **复制同步码**，把同步码发到另一台设备。
2. 另一台设备打开页面后，点 **从同步码恢复**，粘贴并确认即可恢复全部评分、类别和本周菜单。

### 方式2：GitHub 云同步（可选）
1. 打开网页「同步」标签。
2. 填写仓库：`owner/repo`（或粘贴完整仓库链接）。
3. 分支一般填 `main`，文件路径默认 `data/zwcsm-state.json`。
4. 填入个人 GitHub Token（至少有 `repo` 权限）。
5. 点击「保存配置」，再点「上传到云端」。
6. 另一台设备按同样配置并输入同一个 Token，点「从云端拉取」即可拿到同一份数据。

说明：
- 无 token 方式是“复制同步码/恢复同步码”。
- GitHub 方式是直接文件同步，不需要手工改 JSON。
